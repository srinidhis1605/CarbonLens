# CarbonLens
**An AI-Based Website Carbon Impact Analyzer for Sustainable Web Analytics.**

## 📖 About
CarbonLens is an AI-powered web analytics platform that estimates website carbon emissions. It helps developers and business owners understand the hidden environmental costs of their digital presence and provides actionable recommendations to optimize performance and reduce their carbon footprint.

## 🚀 Key Features
- **Carbon Estimation:** Real-time analysis of website resource usage.
- **Sustainability Insights:** AI-generated optimization recommendations powered by OpenAI.
- **Reporting:** Downloadable sustainability reports (PDF/CSV).
- **Dashboard:** Track historical performance and sustainability scores.

## 🛠️ Technology Stack
- **Frontend:** React, Tailwind CSS
- **Backend:** Node.js, Express
- **Database:** PostgreSQL
- **AI Integration:** OpenAI API
- **Containerization:** Docker

## 📋 Functional Modules
1. **Authentication:** Secure JWT-based user management.
2. **Website Analysis:** Automated resource profiling using Puppeteer.
3. **Carbon Estimation:** Mathematical modeling based on Sustainable Web Design principles.
4. **Recommendation Engine:** AI-driven optimization suggestions.
5. **Reporting:** Automated generation of sustainability analytics.
